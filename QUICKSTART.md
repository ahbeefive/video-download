## Local Development (Linux/Mac)

```bash
# 1. Install dependencies
npm install

# 2. Install yt-dlp
pip install -U yt-dlp

# 3. Install ffmpeg
# Ubuntu/Debian:
sudo apt-get install ffmpeg
# Mac:
brew install ffmpeg

# 4. Start server
npm start

# 5. Open browser
http://localhost:3000
```
